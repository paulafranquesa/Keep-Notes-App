# Fulls — notes app / aplicació de notes

*(Català a continuació de l'anglès / Catalan follows English)*

---

## English

**Fulls** is a personal, Google Keep‑style notes app (single HTML file, no install needed) with folders, custom note colors, and a checklist mode.

### Features
- **Folders you create** — name each folder, give it a free‑text "type" label (e.g. *travel*, *task*, *goal*), and pick its color. Folders appear as index‑tab style entries in the sidebar.
- **"All notes" view** — see every note together, regardless of folder.
- **"Unfiled" view** — quickly find notes you haven't assigned to a folder yet.
- **Custom colors per note** — a pastel palette (Sakura, Matcha, Sora, Kuchinashi, Fuji, Momiji, Sumi, Paper), just like Google Keep's color picker.
- **Two note types** — plain text notes, or checklists with tickable items.
- **Pin** important notes to keep them at the top.
- **Search** across titles, text and checklist items.
- **Autosave** — every change is saved automatically as you type/click; nothing to export or import manually.

### How to use it
1. Open `fulls.html` in your browser (as a Claude artifact, or by opening the file directly).
2. Click **"Fes una nota…" / "Take a note…"** at the top to create a note. Choose a color, optionally assign a folder, and toggle checklist mode with **☑ Llista**.
3. Click **"+ Nova carpeta"** in the sidebar to create a folder: give it a name, an optional type label, and a color.
4. Click the **⋯** on a folder tab to rename, recolor, or delete it. Deleting a folder does not delete its notes — they move to "Sense classificar / Unfiled".
5. Click any note to open it, edit title/content/checklist items, change its color or folder, then click **Tanca / Close** (or click outside, or press Esc) to save.
6. Use the search box to filter notes by keyword; it searches inside all folders.

### Where your data lives
All notes and folders are saved to this artifact's own private storage, tied to your account — nothing is uploaded elsewhere and nothing is shared with other users. Data persists the next time you reopen this same artifact.

### Note
This was built as a personal tool based on a screenshot of an existing Google Keep board, so a few starter folders (Viatges, Tasques, Objectius, Personal) are included as an example — feel free to rename or delete them.

## Català

**Fulls** és una aplicació de notes personal, a l'estil de Google Keep (un sol fitxer HTML, sense necessitat d'instal·lar res), amb carpetes, colors personalitzats per nota i mode de llista de tasques.

### Funcionalitats
- **Carpetes que crees tu** — posa nom a cada carpeta, afegeix-hi una etiqueta de "tipus" lliure (p. ex. *viatge*, *tasca*, *objectiu*) i tria el seu color. Les carpetes apareixen a la barra lateral amb aspecte de pestanyes d'índex de llibreta.
- **Vista "Totes les notes"** — veu totes les notes juntes, independentment de la carpeta.
- **Vista "Sense classificar"** — troba ràpidament les notes que encara no has assignat a cap carpeta.
- **Colors personalitzats per nota** — una paleta pastel (Sakura, Matcha, Sora, Kuchinashi, Fuji, Momiji, Sumi, Paper), igual que el selector de colors de Google Keep.
- **Dos tipus de nota** — notes de text normal, o llistes de tasques amb elements marcables.
- **Fixar** notes importants perquè quedin sempre a dalt.
- **Cercador** que busca en títols, text i elements de les llistes.
- **Desat automàtic** — cada canvi es desa sol mentre escrius o fas clic; no cal exportar ni importar res.

### Com utilitzar-la
1. Obre `fulls.html` al navegador (com a artefacte de Claude, o obrint el fitxer directament).
2. Fes clic a **"Fes una nota…"** a la part de dalt per crear-ne una. Tria un color, assigna-hi una carpeta (opcional) i activa el mode llista amb **☑ Llista**.
3. Fes clic a **"+ Nova carpeta"** a la barra lateral per crear-ne una: dona-li un nom, un tipus opcional i un color.
4. Fes clic al **⋯** d'una pestanya de carpeta per canviar-ne el nom, el color o eliminar-la. Si elimines una carpeta, les seves notes no s'esborren — passen a "Sense classificar".
5. Fes clic a qualsevol nota per obrir-la, editar el títol, el contingut o els elements de la llista, canviar-ne el color o la carpeta, i després fes clic a **Tanca** (o fora del quadre, o prem Esc) per desar-ho.
6. Utilitza el cercador per filtrar notes per paraula clau; busca dins de totes les carpetes.

### On es guarden les teves dades
Totes les notes i carpetes es desen a l'emmagatzematge privat d'aquest artefacte, vinculat al teu compte — no es puja res enlloc més ni es comparteix amb altres usuaris. Les dades es mantenen la propera vegada que obris aquest mateix artefacte.

### Nota
Aquesta app es va crear com a eina personal a partir d'una captura d'un tauler existent de Google Keep, per això inclou algunes carpetes d'exemple (Viatges, Tasques, Objectius, Personal) — pots canviar-los el nom o eliminar-les quan vulguis.
